## 前端项目模板

### 后台管理系统分支说明

#### 1.qifu-cms-vuecli
-   说明：qifu公司定制化、无菜单、多页
-   项目框架：vue-cli5 + vue3 + pinia + element-plus2 + mockjs
-   风格：绿色、紧凑型

#### 2.qifu-cms-vuecli-template
-   说明：qifu-cms-vuecli项目脚手架的模板

#### 3. qifu-cms-vite
-   说明：qifu公司定制化、无菜单、多页
-   项目框架：vite + vue3 + pinia + element-plus2 + mockjs

#### 4. qifu-cms-vite-template
